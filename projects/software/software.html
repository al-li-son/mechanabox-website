Software

Invisible Map & Map Creator

iOS apps aimed to assist blind and visually impaired (BVI) people with indoor navigation. I worked on this project through the Olin College Crowdsourcing and Machine Learning (OCCaM) lab, an assistive technology focused lab run by Paul Ruvolo. My main goals for the research experience were to learn app development (both UI and back-end), understand theoretical concepts behind SLAM graph optimization problems, and learn the process of working closely with co-designers to create the best experience for the users of the app in the BVI community.

Point of No Return

A 2D infinite world survival thriller game programmed using Python's pygame library. I worked on this game as a final project for Software Design. I primarily practiced skills in designing code architecture (Model View Controller structure) through this project.

Reddit Sentiment Analysis

A data analysis project examining whether there is a connection between the calculated sentiment of an initial comment on a Reddit post and the sentiment of the subsequent replies. I worked on this project as a midterm project for Software Design. The goal of the project was to practice skills in using APIs to obtain and organize data and using Python and Jupyter Notebooks to manipulate, visualize, and analyze the data.

Handwritten Digit Recognition Algorithm

An algorithm written in Matlab to identify handwritten digits in an image and also perform rudimentary arithmetic. Our algorithm is a slight modification of the Eigenfaces facial recognition algorithm. This project was created for Quantitative Engineering Analysis 1. My primary goals with the project were to see whether Eigenfaces could be successfully modified to apply to a dataset that did not contain faces and to advance on the algorithm, and to add the additional feature of splitting up an image with multiple digits and constructing an equation from it.

Julia Set Visualization

An interactive Java program to visualize the Julia Set. It includes menu objects to manipulate different parameters of the Julia Set equation, as as for panning and zooming across the visualization. The project was a way to get to know fractals as a mathematical concept on a basic level, while getting more familiar with Java GUI components. This was a project for Data Structures.

Minesweeper

A version of the popular classic Windows game Minesweeper coded in Java. It includes the standard three modes and has the timer and mines counter, as well as additional options to change the icons of the game. The main goal of this project was to get more familiar with basic game design and to develop skills with Java GUI elements. This was created for Data Structures.

3D Maze Game

A 3D maze game where a player navigates through different levels, collecting items with the goal of making it to the end. The main goal of this project was to familiarize myself with Java GUI and challenge myself with mimicking perspective in 2D. This was created for Data Structures.
